Missioni Web Application

Django Python Bootstrap Web Development

About This Project

Missioni is a web application designed to simplify the process of managing expenses during university missions and requesting reimbursements from the organization.

The application offers a simple and intuitive tool for the semi-automatic compilation of reimbursement request documents, based on expenses made during missions.

Originally developed by the AImage Lab at Unimore, this project was part of my bachelor's thesis work under the supervision of Prof. Federico Bolelli, where I contributed by adding new features and optimizing existing workflows to improve the overall user experience.

My Contributions

As part of my bachelor's thesis, I implemented three main areas of improvements to the application:

1. Responsive Design and Mobile Adaptation

I adapted the entire interface to ensure optimal visualization of all pages on mobile devices. This included implementing responsive layouts for profile data entry forms, mission forms, mission lists, and mission reports.

2. Image Upload Service for Mission Expenses

I integrated a functionality that allows users to attach receipt images, facilitating effective documentation of expenses made during missions. This required database updates, form modifications, and view changes to handle image uploads and storage.

3. Data Migration from Old to New Format

I developed Python scripts to migrate existing data from the old JSON-based format to the new database structure, ensuring continuity and integrity of existing information. This migration process handled meals, accommodations, conferences, and other expenses.

Technical Implementation

The modifications involved working with Django's Model-View-Template architecture, implementing Bootstrap components for responsive design, and utilizing Django formsets for handling multiple expense entries. The image upload functionality was integrated using Django's file handling capabilities.
